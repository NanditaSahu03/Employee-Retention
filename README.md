# Employee-Retention

The HR department of a multinational company would like to understand the reasons for premature exit of experienced employees by building a machine learning model to predict the 
exit of employees.

Used Classification Algorithms (Accuracy):

a. Logistic Regression ==> 76%

b. Support Vector Machine(Linear) ==> 76%

c. Kernel SVM ==> 90%

d. K Nearest Neighbour ==> 95%

e. Naive Bayes ==> 73%

f. Decision Tree Classifier ==> 96%

g. Random Forest ==> 96%

Decision Tree Classifier and Random Forest gave great results for this case study .
